# UMAP Optimization Animation

English version. Chinese version: [README_CN.md](README_CN.md)

![UMAP neighborhood optimization demo](umap-demo.png)

Run:

```bash
python3 umap/main.py
```

This demo does not call `umap-learn`. It implements a k-nearest-neighbor graph, fuzzy graph weights, and low-dimensional attraction/repulsion optimization.
