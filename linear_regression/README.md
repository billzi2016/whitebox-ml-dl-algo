# Linear Regression Gradient Descent Animation

English version. Chinese version: [README_CN.md](README_CN.md)

![Linear Regression gradient descent demo](lr-demo.png)

Run:

```bash
python3 linear_regression/main.py
```

This demo does not call `sklearn.linear_model.LinearRegression`. It implements MSE, gradients, and gradient descent by hand. The red line is the current fit, and gray vertical segments show residuals for sampled points.
